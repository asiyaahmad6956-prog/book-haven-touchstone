# Book Haven Bookstore Touchstone Project

This is a static HTML/CSS/JavaScript website for Sophia Learning Introduction to Web Development Touchstone 3.

## Files
- `index.html` - Home page
- `gallery.html` - Book gallery and shopping cart
- `about.html` - About page, hours table, and contact/custom request form
- `events.html` - Custom fourth page with reading events and customer feedback
- `style.css` - External stylesheet
- `script.js` - JavaScript functionality and web storage
- `images/` - Original SVG image placeholders

## JavaScript Features
- Responsive menu button
- Active navigation link
- Subscribe feature with email validation
- Add to Cart and View Cart using `sessionStorage`
- Contact/custom order form saved to `localStorage`
- Customer feedback form saved to `localStorage`

## GitHub Pages
Upload all files to a public GitHub repository and enable GitHub Pages from the `main` or `master` branch using the repository root folder.
